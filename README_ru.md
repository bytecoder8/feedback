# Feedback API Service
Сервис управления обратной связью

## Установка
Выполните команды:

`npm i`

`cp .env.example .env`

`docker compose up` (для запуска postgres в docker)

`npx prisma db push`

`npx prisma generate`


## Запуск
`npm run start`

## Документация API
http://localhost:4000/swagger
